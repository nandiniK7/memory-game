Memory Game

This is a simple Memory Card Game built using HTML, CSS, and JavaScript.

The player flips cards to find matching emoji pairs. If two selected cards match, they remain visible. If they do not match, they flip back after a short delay.

## Features

- Card flip functionality
- Matching pair logic
- Shuffle cards randomly
- Restart game button
- Win message after completing the game
- Responsive design for desktop and mobile
- DOM manipulation using JavaScript


## Technologies Used

- HTML
- CSS
- JavaScript


## How to Run

1. Download or clone the project
2. Open the folder in VS Code
3. Run `index.html` using Live Server

## Game Rules

- Click a card to reveal it
- Match two identical cards
- Matched cards stay visible
- Non-matching cards flip back
- Match all cards to win


## Project Structure

- `index.html` → Structure
- `style.css` → Styling
- `script.js` → Game logic


## Result

A fully functional and responsive memory game using JavaScript DOM manipulation.
